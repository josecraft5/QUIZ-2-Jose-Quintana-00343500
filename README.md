# Descripción
Este programa permite al usuario ingresar una cantidad de números por teclado, guardar esos números en una lista y luego ordenarlos de menor a mayor usando el algoritmo Bubble Sort.

El algoritmo Bubble Sort funciona comparando elementos adyacentes de la lista, es decir, compara un elemento con el siguiente. Si el elemento de la izquierda es mayor que el de la derecha, se intercambian de posición. Este proceso se repite varias veces hasta que toda la lista queda ordenada.
